📋 Complete Project Overview
ReturnRack is an AI-powered warehouse returns routing application. It uses a cascaded 4-layer prediction model to classify returned parcels to their appropriate warehouse. The system combines rule-based heuristics, Bayesian inference, gradient boosting, and statistical encoding.

🔗 Layer Architecture & Details
ReturnRack uses cascading prediction where each layer handles increasingly complex patterns. Early layers achieve high accuracy on simple cases.
⚡
Layer 1: Rule Engine
Deterministic
Coverage
65%

Bayesian
Coverage
20%

Probabilistic merchant→warehouse mapping. Uses high-purity merchant associations where P(warehouse|merchant) ≫ 0.95.
Strategy
Bayesian probability with confidence thresholds

Ensemble

🎛️ UI Components
Reusable UI components that compose the ReturnRack dashboard. Each component is self-contained and responsive.
🔍
Search Input
Primary input field for tracking IDs. Supports real-time validation and merchant dropdown selection. Triggers predictions on submit.
📈
Results Panel
Displays prediction output with warehouse, confidence, and breakdown by layer. Shows which layer made the decision.

Recent Scans Table
History table showing last 50 scans. Sortable by timestamp, ID, warehouse. Click to re-predict or export.

Configure confidence thresholds, enable/disable layers, choose between fast/accurate modes. Persist to localStorage.
🏪
Merchant List
Sidebar showing top merchants by volume. Hover to see warehouse breakdown. Click to filter recent scans.
📂
Data Importer
CSV uploader with validation. Shows preview before training. Supports incremental and full retraining modes.
